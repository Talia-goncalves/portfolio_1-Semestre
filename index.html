Olá, Sejam Muito Bem-Vindos Ao Meu Portifólio Web!

  Meu nome é Talia da Silva Gonçalves, tenho 19 anos e atualmente estou cursando o Superior Tecnologo em Análise e desenvolvimento de Sistemas (ADS) na Faculdade Senai Sorocaba.

  Através deste site, vou apresentar a vocês tudo o que aprendi e desenvolvi neste meu primeiro semestre da faculdade, tivemos contato com várias matérias de extrema importância para nosso desenvolvimento na área de Tecnológia.

  

Linguagem de Programação

HTML e CSS

  Ao início do semestre vimos os principais conceitos de HTML e CSS, sempre com teoria e práticas paralelas para fixarmos bem cada conteúdo passado.

  Aprendemos como fazer a estrututa base do HTML, as tags e atributos mais utilizados em códigos, como parágrafos, algumas tags de formatação e estilização em CSS, aprendemos a identificar como é a sintaxe do inicio e do fechamento de cada tag, como fazer para manipular o HTML no CSS, vimos algumas maneiras de organizar os elementos em que criavamois através de divs, etc.

  Nosso primeiro projetinho foi desenvolver com o que haviamos aprendido um curriculo web nos apresentando.

  Logo após este projeto, demos continuidade ao assunto de HTML, vimos o HTML semântico, como é importante ser implementado em nossos código, para ficar um código organizado, mais otimizado e com uma melhor vizibilidade e entendimento.

  E como de costume, ao final de cada aprendizado de um novo conteúdo, criamos um projeto para colocar em prática tudo o que foi aprendido. O projeto em questão se trata de um site simples, aplicando alguns conceitos semânticos

Linguagem JavaScript

  Em seguida tivemos a introdução a linguagem javaScript, nestas aulas aprendemos com conceito fundamentais de todas as linguagens de programação, tais como:

  O que é variável, tipos de dados, a sixtaxe, operadores lógicos e operadores aritméticos, estruturas de repetição e estruturas de decisão, função, arrays etc.

  Após termos conhecimento de todos estes conceitos de HTML, CSS e JavaScript, aprendemos a fazer a aplicação deles em um mesmo projeto. Tivemos alguns exercícios para criar por exemplo calculadoras, então tinhamos que fazer a lógica por trás de cada calculo para que de fato ela funcionasse e depois de termos entendido bem como se fazia, fizemos uma interface grafica aplicando tudo o que foi aprendido em um site que gerava uma interface bonita ao código adicionado aos campos.

Java e Orientação a Objetos

  Tivemos um curso de Java com certificação da Oracle que foi muito importante e que nos trouxe muito conhecimento, foram 3 meses de curso.

  Neste curso vimos a linguagem Java e vimos não somente os conceitos básicos que aprendemos anteriormente com a linguagem JavaScript e alguns outros que não haviamos visto ainda como: Exceptions, tratamento de erros, vimos mais a fundo os Modificadores de Acesso etc. E em principal, vimos conceitos importantíssimos de Programação Orientada a Objetos (POO), vimos todos os conceitos e a aplicação de: Classes, Objetos, Parâmetros, Construtores, Sobrecarga, Métodos Específicos e Métodos de Acesso, e os 4 pilares do POO, a Abstração, o Encapsulamento, a Herança, e por ultimo oPolimorfismo.

  Ao decorrer do curso fizemos exercícios práticos de projetos que faziam parte do curso para aplicarmos passo a passo do que estávamos vendo, mas também fizemos alguns projetos a parte aplicando os conceitos, por exemplo, fizemos um projeto de "Screen Protector" e um joguinho que tivemos como projeto final do curso, passado pelo nosso professor Cainã, que estáva nos instruindo ao decorrer do curso.

Automação Industrial

  O segundo assunto que tivemos após ver linguagem de programação foi automação, no caso, trabalhar com Tinkercad, arduíno e etc.

  Para trabalharmos com as ferramentas do Tinkercad tivemos que utilizar a linguagem C++, e vimos como funcionava LED's, Resistores, botão, placa de ensaio etc.

  E como projeto tivemos que realizar uma aplicação de um semáforo, onde seria duas ruas paralelas, cada uma com um semáforo e ao final das ruas onde elas se encontram, haveria um outro semáforo para pedestres, e quando apertasse o botão, od dois semáforos dos veiculos iriam parar para liberar acesso ao pedestre.
