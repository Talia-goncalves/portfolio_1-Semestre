Linguagem de Programação

HTML e CSS

  Ao início do semestre vimos os principais conceitos de HTML e CSS, sempre com teoria e práticas paralelas para fixarmos bem cada conteúdo passado.

  Aprendemos como fazer a estrutura base do HTML e suas principais tags e atributos, da mesma forma em relação ao CSS, aprendemos a identificar como é a sintaxe do início e do fechamento de cada tag, como fazer para manipular o HTML no CSS etc.

  Nosso primeiro projeto foi criado paralela as explicações que íamos recebendo referente a cada aplicação dos atributos, criei um site simples de receitas, o que me auxiliou a entender como fazer para criar links entre páginas, manusear divs, adicionando imagens e fazendo a estilização geral de um site em si.

  Depois de aprender um pouco mais dos conceitos de HTML e CSS, tivemos um projeto para nos apresentar um pouco melhor e demonstrar como estávamos caminhando referente os nossos aprendizados, e assim elaboramos um currículo, onde coloquei tudo o que aprendi, fiz a navegação entre duas páginas, aqui já entendi melhor como fazer os posicionamentos e separações com as divs.

  Logo após este projeto, demos continuidade ao assunto de HTML, vimos o HTML semântico, como é importante ser implementado em nossos códigos, para ficar um código organizado, mais otimizado e com uma melhor visibilidade e entendimento. E como de costume, ao final de cada aprendizado de um novo conteúdo, criamos um projeto para colocar em prática tudo o que foi aprendido. O projeto em questão se trata de um site simples, aplicando alguns conceitos semânticos.

Linguagem JavaScript

  Quando partimos para linguagem javaScript, aprendemos os conceitos fundamentais de todas as linguagens de programação, tais como: O que é variável, tipos de dados(se são números inteiros ou decimais, se são caracteres ou palavras etc.), a sintaxe, operadores lógicos(utilizados para fazer comparativos) e operadores aritméticos(ou seja como fazemos as operações matemáticas, como soma, divisão, subtração etc), estruturas de repetição(são estruturas de código que te ajudam a repetir um mesmo bloco de código quantas vezes você precisar sem precisar repetir o código. Ex: For, While e Do While) e estruturas de decisão(são estruturas que te dão 2 tipos de resultados em relação as condições apresentadas em seu bloco, verdadeiro ou falso, se verdadeiro faz uma sequencia de código, se falso uma outra sequencia de código. São exemplos deste tipo de estrutura: if, switch), função, arrays(utilizado para armazenas mais de uma valor em uma mesma variável) etc. Após termos conhecimento de todos estes conceitos de HTML, CSS e JavaScript, aprendemos a fazer a aplicação deles em um mesmo projeto.

  Para praticarmos um pouco mais a questão da nossa lógica de programação e os conceitos de dados, e os operadores tivemos alguns exercícios para programar a lógica e os cálculos por trás de cada exercício ou situação matemática, por exemplo, a lógica por trás das calculadoras, então tínhamos que desenvolver a lógica por trás de cada cálculo para que de fato ela funcionasse.

  Após termos entendido bem como se fazia, fizemos uma interface gráfica aplicando tudo o que foi aprendido em um site que gerava uma interface bonita ao código adicionado aos campos específicos de caixa de texto da parte de HTML, e de JavaScript, então após clicar em gerar, o site te levava a uma outra página onde foi acrescentada uma estilização de um CSS já pronto.

  E nosso último projeto utilizando HTML, CSS e javaScript foi uma folha de pagamento, onde fizemos a página estilizada na web, onde tem campos para recebermos informações dos usuários, que seria necessários para fazer os calculos dos impostos e da folha de pagamento de maneira geral, aqui utilizamos o DOM (Document Object Model), para fazer a localização e extração dos dados inseridos pelo usuário e transportar estas informações do HTML para o JS, e assim fazer cada um dos calculos e redirecionar os resultados do JS novamente para campos específicos no HTML para serem expostos na página para o usuário novamente.

Java e Orientação a Objetos

  Ao decorrer deste primeiro semestre da faculdade, tivemos um curso de Java Fundamentals da Oracle, esse curso ajudou muito a entendermos melhor os termos e aplicações da programação, principalmente pelo fato de termos visto e nos aprofundado um pouco em Programação Orientada a Objetos (POO).

  Vimos as bastante a sintaxe de Java e como ela se diferencia da linguagem Java Script que havíamos visto anteriormente, e ali nos foi apresentado os pilares de POO: Abstração, Encapsulamento, Herança e o Polimorfismo. Vimos também além do que havíamos visto nas aulas de javaScript, vimos modificadores de acesso, exceptions, construtores e sobrecargas, objetos, métodos específicos e métodos de acesso, vimos mais a fundo a questão de classes e parâmetros e afins.

  Para colocarmos em prática tudo o que íamos vendo, fizemos um projeto de um Screen Protector, onde usamos todos os conceitos citados acima para desenvolver este projeto, e ainda utilizamos algumas casses e pacotes de bibliotecas nativas da própria linguagem Java. Utilizamos a abstração para fazer com que cada forma/objeto se movimentasse de maneira diferente por exemplo, e fizemos também o mapeamento de teclas para faze com que a logo se movesse da maneira em que o usuário desejasse ao clicar nas setas do teclado.

  Esse curso foi muito importante para compreender de maneira mais prática e detalhada conceitos e funções que são muito utilizadas na programação de maneira geral.
